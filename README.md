## ⚡ Quick Start

1. **Clone the repo**

   ```bash
   git clone https://github.com/Pratham-Prog861/portfolio-2.git
   ```

2. **Install dependencies**

   ```bash
   pnpm install
   ```

3. **Run the development server**

   ```bash
   pnpm dev
   ```

4. **Customize**
   Update `src/data/resume.tsx` to personalize the content.
